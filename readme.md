# 基于springboot的在线小说管理系统1



## 1、项目介绍

基于springboot的在线小说管理系统拥有两种角色：

管理员：会员管理、分类管理、小说管理、评分管理、论坛管理、公告管理

会员：登录注册、下载小说、评分、收藏、发布帖子

## 2、项目技术

后端框架：springboot、Mybatis

前端技术：前端html、后端VUE前后分离

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
